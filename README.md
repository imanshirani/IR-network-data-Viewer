# IR-network-data-Viewer
data Viewer for OSINT dataset of 4,700+ Twitter/X accounts linked to Islamic Republic influence networks. Metadata-only for CIB research &amp; network analysis.

https://github.com/goldenowlosint/Islamic-Republic-Influence-Networks

How to Use :

UnRar "IR network data Viewer.rar" File

Run app.exe

New Features & Improvements:

Pagination Support: Implemented pagination (100 records per page) to improve performance and loading speed.

Row Numbers: Added a row number column to the data table for better tracking.

Clickable Profiles: Users can now double-click on usernames to open their Twitter/X profiles directly.

Bug Fixes: Resolved file loading issues at startup and fixed profile link interactions.

اضافه کردن صفحه‌بندی، شماره ردیف و لینک پروفایل

- اضافه شدن قابلیت صفحه‌بندی (۱۰۰ رکورد در هر صفحه) برای بالا بردن سرعت.
- اضافه شدن ستون شماره ردیف به جدول.
- قابلیت دابل‌کلیک روی نام کاربری برای باز کردن پروفایل توییتر.
